myFlix React

About the Project

This is the React version of front end to the movie API I created. It allows users to sign up, log in, view information about movies, add and delete movies from a list of favorites, and edit or delete their acconut. The main goal of this project was to learn how to code in the popular React framework.

Technologies Used

  • React
  • ReactDOM
  • PropTypes
  • React Redux
  • React Bootstrap
  • React Router DOM
  • Redux
  • Parcel
  • Axios

Features

  • Users can sign up and log in.
  • Users can
  • Specify the number of events shown.
  • Offline functionality.
  • Add an app shortcut to the home screen.
  • View a chart showing number of upcoming events by city
