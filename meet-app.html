Meet App

About the Project

This is an app that uses the Google Calendar API to fetch a list of events. The main goals of building this app was to get a feel for test-driven development and to learn about progressive web apps.

Technologies Used

  • React
  • Recharts
  • Jest
  • Enzyme
  • Cucumber
  • Puppeteer
  • Progressive Web Application
  • Amazon Web Services
  • Atatus
  • Google Calendar API
  • NProgress

Features

  • Filter events by city.
  • Show/hide event details.
  • Specify the number of events shown.
  • Offline functionality.
  • Add an app shortcut to the home screen.
  • View a chart showing number of upcoming events by city
