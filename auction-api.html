Auction API

About the Project

This is an API for an auction website that would allow users to create auctions and bid on them if they create an account. They should still be able to search for and view auctions if they don't have an account. There were numerous goals for this project, one of which was using a SQL database with my website. Refreshing my Javascript skills was another, since as of writing, I have been doing a lot more programming in Python.

I felt like I learned quite a bit with this project. Creating tests after everything else helped give me a greater appreciation for TDD. This was also my first time using either of Mocha or Chai. Creating the database tables and having to readjust them numerous times helped me see the sort of planning and care needed for database creation. I learned a lot more about JWT tokens and how to use better than how I did in the movie API I created. Having to end auctions automatically lead me to learning about CRON tasks and how they were not suited to what I wanted to do.

Features

  • Create an Account
  • Search for items by keyword
  • Create an auction and upload a photo of the item being sold
  • Place a bid on an auction
  • Leave feedback for the buyer/seller of the auction
  • Look up bid history of your own account or of an auction
  • Auctions automatically end after the selected number of days
  • Delete your own account
