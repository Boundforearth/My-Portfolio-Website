Chat App

About the Project

This is a project built with React Native and Expo. It allows users to create a username and chat with friends and family. The goal in creating this app was to learn about building native applications and testing them in virtual machines. It also helped in seeing what some of the different behaviors there are between iOS and Android, and even the differences between different versions of Android.

Technologies Used

  • React Native
  • Expo
  • Expo Image Picker
  • Expo Location
  • Gifted Chat
  • Google Firebase
  • JavaScript
  • Proptypes

Features

  • Users can set a username and background color.
  • Users can chat with friends and family.
  • Users can take a picture and send it over the chat.
  • Users can share photos they have already taken.
  • Users can send their current location.
