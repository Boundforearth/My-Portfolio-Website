myFlix React

About the Project

This is a Pokedex app built in JavaScript that fetches information about Pokemon from the PokeAPI and displays it on the page. Users will then be able to click on a Pokemon which will open up a modal displaying detailed information about that Pokemon. The main goal of this project was to learn JavaScript while also learning how to make requests to APIs.

Technologies Used

  • HTML
  • CSS
  • JavaScript
  • Bootstrap
  • React Bootstrap

Features

  • Users can select which generation of Pokemon they are interested in (up to the 4th)
  • Users can search for a paricular pokemon.
  • Users can click on a Pokemon to get more information about it.
  • Users can go to the next or previous Pokemon from the more information screen.
