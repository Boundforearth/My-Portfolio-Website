Movie API

About the Project

This is an API that is to be used in conjunction with a front end movie app. It has functionality to get that would allow users to sign up and be authenticated to use the app and can get information from the database. The main goal of this project was to learn how create back end code and learn about noSQL databases.

Technologies Used

  • Node
  • Express
  • Cors
  • Bcrypt
  • Jsonwebtoken
  • Mongoose
  • Morgan
  • Passport
  • MongoDB Atlas
  • Heroku
  • JSdoc

Features

  • Allows users to register and login.
  • Allows registered users to view information about movies that are included in the database.
  • Users will have a favorites list from which they can add or delete movies.
  • Users will be able to update their user information
