myFlix Angular

About the Project

This is the Angular version for the front end to the movie API that I created. It allows users to register, sign in, edit their profile, and add movies to their favorites. The main goal of this project was to learn the basics of TypeScript and Angular.

Technologies Used

  • Angular
  • TypeScript
  • RxJS
  • Angular Material
  • TypeDoc
  • SCSS

Features

  • Register a user account and sign in
  • View a list of movies and get more information about the movies such as who directed it.
  • Allows users to add and delete movies from their list of favorites
  • Allows users to edit or delete their account
